Software and database developed

Neurodegenerative Disease Gene Expression database (NDGEx) : A database of differential expression results in 25 neurodegenerative diseases from 155 independent microarray datasets

Cell Death and Survival gene database (CDSG) : A database of 8,380 predicted genes from 175 species linked with literature on cell death and survival

Escherichia coli functional protein-protein interactions (EcoFunPPI) : A database of functional protein-protein interactions in Escherichia coli K12 predicted using a combination of 9 machine learning methods

pub2path : R package to reconstruct biological pathway or process of your interest without a prior knowledge of gene functions from PubMed literature. Access will be given on request since its in final stage of development. For example, above- mentioned CDSG database created using pub2path.
